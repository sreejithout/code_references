# Array List
- We can store any type of data type in an array list.

## Array
```c#
string[] country = new string[3];
country[0] = "USA"; //only string value can be added
country[1] = "Denmark";
country[2] = "Russia";
```
## ArrayList
```c#
//can store different data types
ArrayList arraylist = new ArrayList();
arraylist.Add(3);
arraylist.Add("USA");
arraylist.Add(false);
```
